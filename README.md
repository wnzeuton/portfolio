# Personal Portfolio — Will Nzeuton

My personal portfolio site, built with React and deployed to GitHub Pages.

**Live site:** [wnzeuton.github.io/portfolio](https://wnzeuton.github.io/portfolio)

## Stack

- React 19 + React Router (HashRouter for GitHub Pages compatibility)
- Plain CSS with custom properties
- Google Fonts: Syne + Inter
- Deployed via `gh-pages`

## Running locally

```bash
npm install
npm start
```

## Deploying

```bash
npm run deploy
```

Builds the app and pushes the `build` folder to the `gh-pages` branch.
